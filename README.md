# 🎯 Trabaja.io

> *Idea original y en base al sufrimiento de alguien que también lo vivió.*

---

Cansado de tener un desastre en tus procesos de selección, no saber el status de los procesos, ni tener una retroalimentación constante de cada uno.

Al igual que tú, yo también paso por este problema día con día y sé lo tedioso que puede llegar a ser el no poder controlar de forma más centralizada todo esto.

Por eso estoy desarrollando esta aplicación para poder ayudarte de la manera más óptima en tu proceso de selección y que puedas aumentar tu rendimiento como profesional, sea cual sea el nivel en el que te encuentres: **Senior, Mid o Junior**. Y cualquier rol: **FullStack, Backend, Frontend, QA, DevOps, Mobile**, etc.

Aquí podrás tener un asistente que te ayuda a llevar un seguimiento más estructurado de tus procesos y tu carrera en general, para que cada decisión que tomes sea en base a argumentos sólidos: sobre qué está pasando en tu proceso, en qué puesto te alineas más, qué deberías mejorar para ser más atractivo en el mercado y para tu finalidad.

Olvídate de tener la información mezclada en notas, en hojas dispersas, en una hoja de cálculo, un documento de Word o correos electrónicos dispersos. Ahora podrás contar con tu información más organizada y con alguien que esté detrás de ti para recordarte cosas que quizás estás olvidando — porque claro, somos seres humanos, somos adultos y tenemos más ocupaciones como para tener la mente 100% en esto y no dejar pasar mínimos detalles de una entrevista, un correo, un documento, una reunión, etcétera.

---

## ✨ Funcionalidades — v1

Para la primera versión de la aplicación se busca demostrar que esta situación tiene una salida y que hay mejores formas de hacer frente a este proceso:

- 📋 **Tracker visual** de procesos y postulaciones
- 👻 **Detector de Ghost Jobs** — saber cuándo un proceso de postulación parece haber sido detenido o cancelado
- 🔔 **Seguimiento inteligente** a tus postulaciones con recordatorios
- 📝 **Notas personales** de entrevistas, llamadas, chats con reclutadores, empleados y clientes para apoyar tus procesos o aprendizaje

## 🛠️ Stack tecnológico

Para esta primera versión se decide usar un stack ligero, funcional y sin problemas de escalabilidad, que pueda coexistir con futuras versiones más complejas, seguras y blindadas.

**Backend**
- Java · Spring Boot · Spring Security · Spring Data JPA (Hibernate)

**Frontend**
- Vite · ReactJS · Tailwind CSS

**AI & Automatización**
- n8n · Gemini Flash

**Base de datos**
- PostgreSQL · PgAdmin

**DevOps**
- Docker
